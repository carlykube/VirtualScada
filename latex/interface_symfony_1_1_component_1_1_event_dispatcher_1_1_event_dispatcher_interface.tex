\subsection{Detailed Description}
The \doxyref{Event\+Dispatcher\+Interface}{p.}{interface_symfony_1_1_component_1_1_event_dispatcher_1_1_event_dispatcher_interface} is the central point of \doxyref{Symfony}{p.}{namespace_symfony}\textquotesingle{}s event listener system. Listeners are registered on the manager and events are dispatched through the manager.
