\subsection{Detailed Description}
Handles content fragments represented by special U\+R\+Is.

All \doxyref{U\+R\+L}{p.}{class_u_r_l} paths starting with /\+\_\+fragment are handled as content fragments by this listener.

If throws an Access\+Denied\+Http\+Exception exception if the request is not signed or if it is not an internal sub-\/request.
