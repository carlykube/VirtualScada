\subsection{Detailed Description}
Test should cover all languages mentioned on {\tt http\+://translate.\+sourceforge.\+net/wiki/l10n/pluralforms} and Plural forms mentioned on {\tt http\+://www.\+gnu.\+org/software/gettext/manual/gettext.\+html\#\+Plural-\/forms}.

See also {\tt https\+://developer.\+mozilla.\+org/en/\+Localization\+\_\+and\+\_\+\+Plurals} which mentions 15 rules having a maximum of 6 forms. The mozilla code is also interesting to check for.

As mentioned by chx {\tt http\+://drupal.\+org/node/1273968} we can cover all by testing number from 0 to 199

The goal to cover all languages is to far fetched so this test case is smaller.
