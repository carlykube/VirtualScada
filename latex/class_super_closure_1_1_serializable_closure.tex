\subsection{Detailed Description}
This class acts as a wrapper for a closure, and allows it to be serialized.

With the combined power of the Reflection A\+P\+I, code parsing, and the infamous {\ttfamily eval()} function, you can serialize a closure, unserialize it somewhere else (even a different \doxyref{P\+H\+P}{p.}{namespace_p_h_p} process), and execute it.

Unserializes the closure.

Unserializes the closure\textquotesingle{}s data and recreates the closure using a simulation of its original context. The used variables (context) are extracted into a fresh scope prior to redefining the closure. The closure is also rebound to its former object and scope.
