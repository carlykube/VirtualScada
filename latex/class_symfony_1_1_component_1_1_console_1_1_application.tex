\subsection{Detailed Description}
An \doxyref{Application}{p.}{class_symfony_1_1_component_1_1_console_1_1_application} is the container for a collection of commands.

It is the main entry point of a \doxyref{Console}{p.}{namespace_symfony_1_1_component_1_1_console} application.

This class is optimized for a standard C\+L\+I environment.

Usage\+: \begin{DoxyVerb}$app = new Application('myapp', '1.0 (stable)');
$app->add(new SimpleCommand());
$app->run();
\end{DoxyVerb}
