\subsection{Detailed Description}
\doxyref{A}{p.}{class_a} \doxyref{Shell}{p.}{class_symfony_1_1_component_1_1_console_1_1_shell} wraps an \doxyref{Application}{p.}{class_symfony_1_1_component_1_1_console_1_1_application} to add shell capabilities to it.

Support for history and completion only works with a \doxyref{P\+H\+P}{p.}{namespace_p_h_p} compiled with readline support (either --with-\/readline or --with-\/libedit)
