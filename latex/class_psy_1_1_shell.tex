Invoke a \doxyref{Psy}{p.}{namespace_psy} \doxyref{Shell}{p.}{class_psy_1_1_shell} from the current context.

For example\+: \begin{DoxyVerb}foreach ($items as $item) {
    \Psy\Shell::debug(get_defined_vars());
}
\end{DoxyVerb}


If you would like your shell interaction to affect the state of the current context, you can extract() the values returned from this call\+: \begin{DoxyVerb}foreach ($items as $item) {
    extract(\Psy\Shell::debug(get_defined_vars()));
    var_dump($item); // will be whatever you set $item to in Psy Shell
}
\end{DoxyVerb}


Optionally, supply an object as the {\ttfamily \$bind} parameter. This determines the value {\ttfamily \$this} will have in the shell, and sets up class scope so that private and protected members are accessible\+: \begin{DoxyVerb}class Foo {
    function bar() {
        \Psy\Shell::debug(get_defined_vars(), $this);
    }
}
\end{DoxyVerb}


This only really works in \doxyref{P\+H\+P}{p.}{namespace_p_h_p} 5.\+4+ and H\+H\+V\+M 3.\+5+, so upgrade already.
