\subsection{Detailed Description}
\doxyref{Response}{p.}{class_symfony_1_1_component_1_1_http_foundation_1_1_response} represents an H\+T\+T\+P response in J\+S\+O\+N format.

Note that this class does not force the returned J\+S\+O\+N content to be an object. It is however recommended that you do return an object as it protects yourself against X\+S\+S\+I and J\+S\+O\+N-\/\+Java\+Script Hijacking.
