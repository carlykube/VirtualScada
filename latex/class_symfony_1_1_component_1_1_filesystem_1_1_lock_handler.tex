\subsection{Detailed Description}
\doxyref{Lock\+Handler}{p.}{class_symfony_1_1_component_1_1_filesystem_1_1_lock_handler} class provides a simple abstraction to lock anything by means of a file lock.

\doxyref{A}{p.}{class_a} locked file is created based on the lock name when calling \doxyref{lock()}{p.}{class_symfony_1_1_component_1_1_filesystem_1_1_lock_handler_ae370e38547e4f437ceac857c010d8061}. \doxyref{Other}{p.}{namespace_other} lock handlers will not be able to lock the same name until it is released (explicitly by calling \doxyref{release()}{p.}{class_symfony_1_1_component_1_1_filesystem_1_1_lock_handler_ad575b496528688fd44a59dd58c18060a} or implicitly when the instance holding the lock is destroyed).
