\subsection*{Installation}

To add this package as a local, per-\/project dependency to your project, simply add a dependency on {\ttfamily sebastian/diff} to your project\textquotesingle{}s {\ttfamily composer.\+json} file. Here is a minimal example of a {\ttfamily composer.\+json} file that just defines a dependency on \doxyref{Diff}{p.}{class_diff}\+: \begin{DoxyVerb}{
    "require": {
        "sebastian/diff": "*"
    }
}
\end{DoxyVerb}


\subsubsection*{Usage}

The {\ttfamily Differ} class can be used to generate a textual representation of the difference between two strings\+:

```php use \doxyref{Sebastian\+Bergmann}{p.}{namespace_sebastian_bergmann};

\$differ = new Differ; print \$differ-\/$>$diff(\textquotesingle{}foo\textquotesingle{}, \textquotesingle{}bar\textquotesingle{}); ```

The code above yields the output below\+: \begin{DoxyVerb}--- Original
+++ New
@@ @@
-foo
+bar
\end{DoxyVerb}


The {\ttfamily Parser} class can be used to parse a unified diff into an object graph\+:

\$diff = \$git-\/$>$get\+Diff( \textquotesingle{}948a1a07768d8edd10dcefa8315c1cbeffb31833\textquotesingle{}, \textquotesingle{}c07a373d2399f3e686234c4f7f088d635eb9641b\textquotesingle{} );

\$parser = new Parser;
