\subsection{Detailed Description}
Validate that the old-\/style constructor function is not static.

As of \doxyref{P\+H\+P}{p.}{namespace_p_h_p} 5.\+3.\+3, methods with the same name as the last element of a namespaced class name will no longer be treated as constructor. This change doesn\textquotesingle{}t affect non-\/namespaced classes.

Validation of the \+\_\+\+\_\+construct method ensures the \doxyref{P\+H\+P}{p.}{namespace_p_h_p} Parser.
