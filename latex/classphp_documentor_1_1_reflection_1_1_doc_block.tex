Returns whether this \doxyref{Doc\+Block}{p.}{classphp_documentor_1_1_reflection_1_1_doc_block} is the start of a \doxyref{Template}{p.}{namespace_template} section.

\doxyref{A}{p.}{class_a} Docblock may serve as template for a series of subsequent Doc\+Blocks. This is indicated by a special marker ({\ttfamily \#@+}) that is appended directly after the opening {\ttfamily /$\ast$$\ast$} of a \doxyref{Doc\+Block}{p.}{classphp_documentor_1_1_reflection_1_1_doc_block}.

An example of such an opening is\+:

``` /$\ast$$\ast$\#@+
\begin{DoxyItemize}
\item \doxyref{My}{p.}{namespace_my} \doxyref{Doc\+Block}{p.}{classphp_documentor_1_1_reflection_1_1_doc_block}
\item / ```
\end{DoxyItemize}

The description and tags (not the summary!) are copied onto all subsequent Doc\+Blocks and also applied to all elements that follow until another \doxyref{Doc\+Block}{p.}{classphp_documentor_1_1_reflection_1_1_doc_block} is found that contains the closing marker ({\ttfamily \#@-\/}).
