\subsection{Detailed Description}
This Logger can be used to avoid conditional log calls

Logging should always be optional, and if no logger is provided to your library creating a \doxyref{Null\+Logger}{p.}{class_psr_1_1_log_1_1_null_logger} instance to have something to throw logs at is a good way to avoid littering your code with {\ttfamily if (\$this-\/$>$logger) \{ \}} blocks.
