\subsection*{About}

\subsection*{Installation}

Download the {\ttfamily psysh} phar to install\+:

``` wget psysh.\+org/psysh chmod +x psysh ./psysh ```

It\textquotesingle{}s even awesomer if you put it somewhere in your system path (like {\ttfamily /usr/local/bin} or {\ttfamily $\sim$/bin})!

Psy\+S\+H {\tt is available via Composer}, so you can use it in your project as well\+:

``` composer require psy/psysh\+: ./vendor/bin/psysh ```

Or you can use by checking out the the repository directly\+:

``` git clone {\tt https\+://github.\+com/bobthecow/psysh.\+git} cd psysh ./bin/psysh ```
