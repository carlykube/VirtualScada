There are multiple ways to include the \doxyref{P\+H\+P}{p.}{namespace_p_h_p} parser into your project\+:

\subsection*{Installing via \doxyref{Composer}{p.}{namespace_composer} }

Run the following command inside your project\+: \begin{DoxyVerb}php composer.phar require nikic/php-parser
\end{DoxyVerb}


If you haven\textquotesingle{}t installed {\tt Composer} yet, you can do so using\+: \begin{DoxyVerb}curl -s http://getcomposer.org/installer | php
\end{DoxyVerb}


\subsection*{Installing as a Git Submodule }

Run the following command to install the parser into the {\ttfamily vendor/\+P\+H\+P-\/\+Parser} folder\+: \begin{DoxyVerb}git submodule add git://github.com/nikic/PHP-Parser.git vendor/PHP-Parser
\end{DoxyVerb}


\subsection*{Installing from the Zip-\/ or Tarball }

Download the latest version from {\tt the download page}, unpack it and move the files somewhere into your project.
