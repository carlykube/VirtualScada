\subsection{Detailed Description}
Allows filtering of a controller callable.

You can call \doxyref{get\+Controller()}{p.}{class_symfony_1_1_component_1_1_http_kernel_1_1_event_1_1_filter_controller_event_aa8b89e0bad51878addc1300cd3e95b5c} to retrieve the current controller. With \doxyref{set\+Controller()}{p.}{class_symfony_1_1_component_1_1_http_kernel_1_1_event_1_1_filter_controller_event_aa4f9c9e545ef3a90856d30de93fe5e09} you can set a new controller that is used in the processing of the request.

Controllers should be callables.
