\subsection{Detailed Description}
\doxyref{Response\+Cache\+Strategy}{p.}{class_symfony_1_1_component_1_1_http_kernel_1_1_http_cache_1_1_response_cache_strategy} knows how to compute the \doxyref{Response}{p.}{class_response} cache H\+T\+T\+P header based on the different response cache headers.

This implementation changes the master response T\+T\+L to the smallest T\+T\+L received or force validation if one of the surrogates has validation cache strategy.
