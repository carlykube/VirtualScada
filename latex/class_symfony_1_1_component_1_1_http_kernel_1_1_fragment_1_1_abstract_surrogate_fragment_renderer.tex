Note that if the current \doxyref{Request}{p.}{class_request} has no surrogate capability, this method falls back to use the inline rendering strategy.

Additional available options\+:


\begin{DoxyItemize}

\item comment\+: a comment to add when returning the surrogate tag
\end{DoxyItemize}

Note, that not all surrogate strategies support all options. For now \textquotesingle{}alt\textquotesingle{} and \textquotesingle{}comment\textquotesingle{} are only supported by E\+S\+I.
