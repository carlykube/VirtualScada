\subsection{Detailed Description}
\doxyref{Console\+Output}{p.}{class_symfony_1_1_component_1_1_console_1_1_output_1_1_console_output} is the default class for all C\+L\+I output. It uses S\+T\+D\+O\+U\+T.

This class is a convenient wrapper around {\ttfamily \doxyref{Stream\+Output}{p.}{class_symfony_1_1_component_1_1_console_1_1_output_1_1_stream_output}}. \begin{DoxyVerb}$output = new ConsoleOutput();
\end{DoxyVerb}


This is equivalent to\+: \begin{DoxyVerb}$output = new StreamOutput(fopen('php://stdout', 'w'));
\end{DoxyVerb}
