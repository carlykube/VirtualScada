\subsection{Detailed Description}
\doxyref{Annotation\+Class\+Loader}{p.}{class_symfony_1_1_component_1_1_routing_1_1_loader_1_1_annotation_class_loader} loads routing information from a \doxyref{P\+H\+P}{p.}{namespace_p_h_p} class and its methods.

You need to define an implementation for the get\+Route\+Defaults() method. Most of the time, this method should define some \doxyref{P\+H\+P}{p.}{namespace_p_h_p} callable to be called for the route (a controller in M\+V\+C speak).

The  annotation can be set on the class (for global parameters), and on each method.

The  annotation main value is the route path. The annotation also recognizes several parameters\+: requirements, options, defaults, schemes, methods, host, and name. The name parameter is mandatory. Here is an example of how you should be able to use it\+: \begin{DoxyVerb}/**
 * @Route("/Blog")
 * /
class Blog
{
    /**
     * @Route("/", name="blog_index")
     * /
    public function index()
    {
    }

    /**
     * @Route("/{id}", name="blog_post", requirements = {"id" = "\d+"})
     * /
    public function show()
    {
    }
}
\end{DoxyVerb}
