\subsection{Detailed Description}
Eases the testing of console applications.

When testing an application, don\textquotesingle{}t forget to disable the auto exit flag\+: \begin{DoxyVerb}$application = new Application();
$application->setAutoExit(false);
\end{DoxyVerb}
