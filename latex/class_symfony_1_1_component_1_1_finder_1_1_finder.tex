\subsection{Detailed Description}
\doxyref{Finder}{p.}{class_symfony_1_1_component_1_1_finder_1_1_finder} allows to build rules to find files and directories.

It is a thin wrapper around several specialized iterator classes.

All rules may be invoked several times.

All methods return the current \doxyref{Finder}{p.}{class_symfony_1_1_component_1_1_finder_1_1_finder} object to allow easy chaining\+:

\$finder = \doxyref{Finder\+::create()}{p.}{class_symfony_1_1_component_1_1_finder_1_1_finder_a239b1c70258014a86569483c2d009de6}-\/$>$\doxyref{files()}{p.}{class_symfony_1_1_component_1_1_finder_1_1_finder_a26f846b09db5e10b30498e97e4ee28ed}-\/$>$name(\textquotesingle{}$\ast$.php\textquotesingle{})-\/$>$in({\bfseries D\+I\+R});
