\subsection*{Installation}

To add Recursion Context as a local, per-\/project dependency to your project, simply add a dependency on {\ttfamily sebastian/recursion-\/context} to your project\textquotesingle{}s {\ttfamily composer.\+json} file. Here is a minimal example of a {\ttfamily composer.\+json} file that just defines a dependency on Recursion Context 1.\+0\+: \begin{DoxyVerb}{
    "require": {
        "sebastian/recursion-context": "~1.0"
    }
}\end{DoxyVerb}
