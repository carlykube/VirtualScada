\subsection{Detailed Description}
The object behaviour is the default base class for specification.

Most specs will extend this class directly.

Its responsibility is to proxy method calls to \doxyref{Php\+Spec}{p.}{namespace_php_spec} caller which will wrap the results into \doxyref{Php\+Spec}{p.}{namespace_php_spec} subjects. This results will then be able to be matched against expectations.
