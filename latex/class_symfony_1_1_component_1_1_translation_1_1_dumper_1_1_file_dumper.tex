\subsection{Detailed Description}
\doxyref{File\+Dumper}{p.}{class_symfony_1_1_component_1_1_translation_1_1_dumper_1_1_file_dumper} is an implementation of \doxyref{Dumper\+Interface}{p.}{interface_symfony_1_1_component_1_1_translation_1_1_dumper_1_1_dumper_interface} that dump a message catalogue to file(s). Performs backup of already existing files.

\item path (mandatory)\+: the directory where the files should be saved
