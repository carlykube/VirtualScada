\subsection{Detailed Description}
Acts as a marker and a data holder for a \doxyref{Controller}{p.}{namespace_symfony_1_1_component_1_1_http_kernel_1_1_controller}.

Some methods in \doxyref{Symfony}{p.}{namespace_symfony} accept both a U\+R\+I (as a string) or a controller as an argument. In the latter case, instead of passing an array representing the controller, you can use an instance of this class.
