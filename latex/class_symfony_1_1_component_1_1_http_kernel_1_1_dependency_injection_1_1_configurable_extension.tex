\subsection{Detailed Description}
This extension sub-\/class provides first-\/class integration with the Config/\+Definition \doxyref{Component}{p.}{namespace_symfony_1_1_component}.

You can use this as base class if you

a) use the Config/\+Definition component for configuration b) your configuration class is named \char`\"{}\+Configuration\char`\"{} and c) the configuration class resides in the \doxyref{Dependency\+Injection}{p.}{namespace_symfony_1_1_component_1_1_http_kernel_1_1_dependency_injection} sub-\/folder
