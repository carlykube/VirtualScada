\subsection{Detailed Description}
\doxyref{Proc\+Output\+Pager}{p.}{class_psy_1_1_output_1_1_proc_output_pager} class.

\doxyref{A}{p.}{class_a} \doxyref{Proc\+Output\+Pager}{p.}{class_psy_1_1_output_1_1_proc_output_pager} instance wraps a regular Stream\+Output\textquotesingle{}s stream. Rather than writing directly to the stream, it shells out to a pager process and gives that process the stream as stdout. This means regular $\ast$nix commands like {\ttfamily less} and {\ttfamily more} can be used to page large amounts of output.
