Handles a record.

All records may be passed to this method, and the handler should discard those that it does not want to handle.

The return value of this function controls the bubbling process of the handler stack. Unless the bubbling is interrupted (by returning true), the \doxyref{Logger}{p.}{class_monolog_1_1_logger} class will keep on calling further handlers in the stack with a given log record.


\begin{DoxyParams}[1]{Parameters}

\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Boolean true means that this handler handled the record, and that bubbling is not permitted. false means the record was either not processed or that this handler allows bubbling. 
\end{DoxyReturn}

Checks whether the given record will be handled by this handler.

This is mostly done for performance reasons, to avoid calling processors for nothing.

Handlers should still check the record levels within \doxyref{handle()}{p.}{interface_monolog_1_1_handler_1_1_handler_interface_ad54b568c673b7b4fc724268a54313603}, returning false in \doxyref{is\+Handling()}{p.}{interface_monolog_1_1_handler_1_1_handler_interface_abbfbbcd3430dea8422ecd3c725737758} is no guarantee that \doxyref{handle()}{p.}{interface_monolog_1_1_handler_1_1_handler_interface_ad54b568c673b7b4fc724268a54313603} will not be called, and \doxyref{is\+Handling()}{p.}{interface_monolog_1_1_handler_1_1_handler_interface_abbfbbcd3430dea8422ecd3c725737758} might not be called for a given record.
