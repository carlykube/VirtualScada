\subsection{Detailed Description}
\doxyref{Number\+Comparator}{p.}{class_symfony_1_1_component_1_1_finder_1_1_comparator_1_1_number_comparator} compiles a simple comparison to an anonymous subroutine, which you can call with a value to be tested again.

Now this would be very pointless, if Number\+Compare didn\textquotesingle{}t understand magnitudes.

The target value may use magnitudes of kilobytes (k, ki), megabytes (m, mi), or gigabytes (g, gi). Those suffixed with an i use the appropriate 2$\ast$$\ast$n version in accordance with the I\+E\+C standard\+: {\tt http\+://physics.\+nist.\+gov/cuu/\+Units/binary.\+html}

Based on the Perl Number\+::\+Compare module.
