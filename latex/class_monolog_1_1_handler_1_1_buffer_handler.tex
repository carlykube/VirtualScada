\subsection{Detailed Description}
Buffers all records until closing the handler and then pass them as batch.

This is useful for a \doxyref{Mail\+Handler}{p.}{class_monolog_1_1_handler_1_1_mail_handler} to send only one mail per request instead of sending one per log message.
