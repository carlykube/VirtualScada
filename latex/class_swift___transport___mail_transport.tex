\subsection{Detailed Description}
Sends Messages using the mail() function.

It is advised that users do not use this transport if at all possible since a number of plugin features cannot be used in conjunction with this transport due to the internal interface in \doxyref{P\+H\+P}{p.}{namespace_p_h_p} itself.

The level of error reporting with this transport is incredibly weak, again due to limitations of \doxyref{P\+H\+P}{p.}{namespace_p_h_p}\textquotesingle{}s internal mail() function. You\textquotesingle{}ll get an all-\/or-\/nothing result from sending.
