\subsection{Detailed Description}
\doxyref{A}{p.}{class_a} \doxyref{Route\+Collection}{p.}{class_symfony_1_1_component_1_1_routing_1_1_route_collection} represents a set of \doxyref{Route}{p.}{class_symfony_1_1_component_1_1_routing_1_1_route} instances.

When adding a route at the end of the collection, an existing route with the same name is removed first. So there can only be one route with a given name.
