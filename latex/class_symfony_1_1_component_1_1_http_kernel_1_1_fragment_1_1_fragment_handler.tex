\subsection{Detailed Description}
Renders a U\+R\+I that represents a resource fragment.

This class handles the rendering of resource fragments that are included into a main resource. The handling of the rendering is managed by specialized renderers.

This listener works in 2 modes\+:


\begin{DoxyItemize}
\item 2.\+3 compatibility mode where you must call set\+Request whenever the \doxyref{Request}{p.}{class_request} changes.
\item 2.\+4+ mode where you must pass a Request\+Stack instance in the constructor.
\end{DoxyItemize}
