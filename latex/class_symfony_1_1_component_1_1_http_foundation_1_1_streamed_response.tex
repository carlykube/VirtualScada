\subsection{Detailed Description}
\doxyref{Streamed\+Response}{p.}{class_symfony_1_1_component_1_1_http_foundation_1_1_streamed_response} represents a streamed H\+T\+T\+P response.

\doxyref{A}{p.}{class_a} \doxyref{Streamed\+Response}{p.}{class_symfony_1_1_component_1_1_http_foundation_1_1_streamed_response} uses a callback for its content.

The callback should use the standard \doxyref{P\+H\+P}{p.}{namespace_p_h_p} functions like echo to stream the response back to the client. The \doxyref{flush()}{p.}{d3_8min_8js_a818fbba7a975c6edc82859a681127bf2} method can also be used if needed.
