\subsection{Detailed Description}
Day of month field. Allows\+: $\ast$ , / -\/ ? L W

\textquotesingle{}L\textquotesingle{} stands for \char`\"{}last\char`\"{} and specifies the last day of the month.

The \textquotesingle{}W\textquotesingle{} character is used to specify the weekday (Monday-\/\+Friday) nearest the given day. As an example, if you were to specify \char`\"{}15\+W\char`\"{} as the value for the day-\/of-\/month field, the meaning is\+: \char`\"{}the nearest weekday to the 15th of the
month\char`\"{}. So if the 15th is a Saturday, the trigger will fire on Friday the 14th. If the 15th is a Sunday, the trigger will fire on Monday the 16th. If the 15th is a Tuesday, then it will fire on Tuesday the 15th. However if you specify \char`\"{}1\+W\char`\"{} as the value for day-\/of-\/month, and the 1st is a Saturday, the trigger will fire on Monday the 3rd, as it will not \textquotesingle{}jump\textquotesingle{} over the boundary of a month\textquotesingle{}s days. The \textquotesingle{}W\textquotesingle{} character can only be specified when the day-\/of-\/month is a single day, not a range or list of days.
