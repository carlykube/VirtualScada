getConfigFile ( )

Get the current PsySH config file.

If a configFile option was passed to the Configuration constructor, this file will be returned. If not, all possible config directories will be searched, and the first config.php or rc.php file which exists will be returned.

If you're trying to decide where to put your config file, pick 

~/.config/psysh/config.php
