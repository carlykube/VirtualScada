Detailed Description

The following methods use reserved keywords, so their implementation is defined with an underscore and made available with the reserved name through __call() magic.

Builder namespace(string $name) Creates a namespace builder. Builder class(string $name) Creates a class builder. Builder interface(string $name) Creates an interface builder. Builder trait(string $name) Creates a trait builder. Builder function(string $name) Creates a function builder. Builder use(string $name) Creates a namespace/class use builder.
