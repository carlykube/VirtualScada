PHPUnit

PHPUnit is a programmer-oriented testing framework for PHP. It is an instance of the xUnit architecture for unit testing frameworks.

![Latest Stable Version](https://poser.pugx.org/phpunit/phpunit/v/stable.png) ![Build Status](https://travis-ci.org/sebastianbergmann/phpunit.svg?branch=4.6) ![Dependency Status](https://www.versioneye.com/php/phpunit:phpunit/4.6/badge.png) ![Reference Status](https://www.versioneye.com/php/phpunit:phpunit/reference_badge.svg)

Installation

We distribute a PHP Archive (PHAR) that has all required (as well as some optional) dependencies of PHPUnit bundled in a single file: 

wget https://phar.phpunit.de/phpunit.phar
chmod +x phpunit.phar
mv phpunit.phar /usr/local/bin/phpunit

You can also immediately use the PHAR after you have downloaded it, of course: 

wget https://phar.phpunit.de/phpunit.phar
php phpunit.phar

Alternatively, you may use Composer to download and install PHPUnit as well as its dependencies. Please refer to the documentation for details on how to do this.
