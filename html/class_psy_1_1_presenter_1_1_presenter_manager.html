Detailed Description

A Presenter manager service.

Presenters are registered with the PresenterManager, which then delegates value presentation to the most recently registered Presenter capable of presenting that value.

addPresenter ( Presenter  $presenter)

Register a Presenter.

If multiple Presenters are able to present a value, the most recently registered Presenter takes precedence.

If $presenter is already registered, it will be re-registered as the highest precedence Presenter.
